# RedCode 🔴
A custom programming language interpreter built in **Python** using **Pydroid 3**.

## Features
- **Variables:** Use `SET` to store data.
- **Math:** Basic `SUB` and `RANDOM` number generation.
- **Loops:** Repeat commands with `REPEAT`.
- **Colors:** Support for terminal colors (RED, GREEN, BLUE).

## How to Run
1. Install **Pydroid 3** on Android.
2. Download `redcode.py`.
3. Run the script and start typing RedCode commands!

## Example
SET name Tristan
COLOR GREEN
REPEAT 3 PRINT Hello name
